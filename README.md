# 书签管理器 (Bookmarks Manager)

一个功能强大、界面美观的浏览器书签管理扩展。

## ✨ 主要功能

- **动态背景**：每天自动从必应获取美丽的背景图片。
- **强大的书签管理**：
  - 网格视图，清晰展示书签和文件夹。
  - 文件夹导航，支持多级目录。
  - 拖拽移动书签或将其放入文件夹。
  - 右键菜单进行快速编辑和删除。
  - 创建新文件夹。
- **全局搜索**：
  - 快速搜索所有书签。
  - 搜索结果显示书签的完整路径，方便定位。
- **常用网站**：
  - 自动记录并显示最常访问的书签。
  - 可在设置中自定义显示数量。
- **高度可定制的界面**：
  - **背景滤镜**：调整背景的亮度、对比度、饱和度、模糊和色调。
  - **颜色叠加**：为背景添加一层自定义颜色和透明度的叠加。
  - **布局切换**：支持标准、紧凑和宽松三种布局模式。
- **所有设置自动保存**：你的自定义偏好会被保存在本地。

## 🚀 如何安装

由于此扩展未在官方商店发布，你需要通过开发者模式手动安装：

1.  下载或克隆此项目的所有文件到本地文件夹。
2.  打开你的浏览器 (Chrome, Edge, 等)。
3.  在地址栏输入 `chrome://extensions` (Chrome) 或 `edge://extensions` (Edge) 并回车。
4.  在页面右上角，打开 **开发者模式** 开关。
5.  点击 **加载已解压的扩展程序** 按钮。
6.  选择你第一步中保存项目文件的文件夹。
7.  安装完成！你应该会在浏览器的扩展程序列表中看到 **Bookmarks Manager**。

## 🛠️ 如何使用

- **打开主界面**：点击浏览器工具栏上的 **Bookmarks Manager** 图标，即可打开一个新的标签页显示主界面。
- **浏览书签**：像文件浏览器一样点击文件夹进入，或点击“返回上级”按钮回到上一层。
- **移动书签**：直接将书签或文件夹拖拽到目标文件夹上，或者拖拽到“返回上级”按钮上以移动到父文件夹。
- **搜索书签**：在顶部的搜索框中输入关键词，结果会实时显示在下方。
- **编辑/删除**：在书签或文件夹上右键，即可选择编辑或删除。

## 🎨 自定义选项

该扩展提供了丰富的自定义选项，可以通过主界面右侧的 **设置** 面板进行调整：

- **背景滤镜**：
  - **亮度、对比度、饱和度、模糊、色调旋转**：通过滑块实时调整背景图片效果。
  - **颜色叠加**：选择一个颜色并调整其透明度，为背景添加一层蒙版效果。
  - **重置**：一键恢复所有滤镜和颜色设置为默认值。
- **常规设置**：
  - **布局设置**：在标准、紧凑、宽松三种布局之间切换。
  - **最近访问显示项目数**：自定义“常用网站”区域显示的书签数量。
